imr_load_data {RIntaros}R Documentation

Load data from a set of NetCDF files

Description

Load data from a set of NetCDF files

Usage

imr_load_data(dir = ".")

Arguments

dir

Directory where the NetCDF files should be read.

Value

The resulting db-class where the contents of all the NetCDF files are concatenated. The file contains the following variables:

  • Longitude

  • Latitude

  • Depth

  • Temperature

  • Conductivity

  • Salinity

  • Time

  • Profil_id

  • Vessel_name

  • Vessel_id
