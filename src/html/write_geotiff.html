write_geotiff {RIntaros}R Documentation

Write a Geotiff file for each variable in the Db with locator "z"

Description

Write a Geotiff file (colorized or mono band) for each interest variable from the grid Db with locator "z" into the output image directory defined in the environment

Usage

write_geotiff(dbg, colors=NA, colors.files=NA, prefix=NA)

Arguments

dbg

The db-class structure organized as a regular grid.

colors

The colors lists to be used (one by variable). First color of each colorscale will be affected to corresponding minimal value and last color for maximal value. A colorscale is a liste of text colors (hexadecimal or plain text). It can be for example the colorscale returned by rg.colors. Not used if colors.file is defined.

colors.files

List of colorscale text pre-existing files to be used (one by variable).

Prefix

Prefix to be append to the front of each Geotiff file name.

Value

Unused
