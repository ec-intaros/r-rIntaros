intaros.load.environ {RIntaros}R Documentation

Get the value of a parameter from Intaros Environment

Description

Get the value of a parameter from Intaros Environment

Usage

intaros.load.environ(filename = ".Intaros.Environ", verbose = FALSE)

Arguments

filename

Name given to the Intaros Environment

verbose

Verbose flag

Value

This function returns a list containing the following items:

  • Long_lim: range of longitude of the offset

  • Lat_lim: range of latitude of the offset

  • Flag_file: flag specifying if the figures are saved on disk

  • Image_dir: Name of the directory where the image files are stored (if 'Flag_file' is TRUE)
